Welcome to the OpenFlap Flap Generator!

This tool allows you generate a set of SVG files required to produce the flap PCB's.

Additionally, you can tweak each glyph's position and size, or even upload custom glyphs.

Once the you have downloaded the ZIP file containing all SVG's, you can convert them to gerbers using the svg_to_gerber.sh utility in this repository.
