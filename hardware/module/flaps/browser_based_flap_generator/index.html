Welcome to the OpenFlap Flap Generator!

This tool allows you to generate a set of SVG files required to produce the flap PCB's.

Additionally, you can tweak each glyph's position and size.

Once the you have downloaded the ZIP file containing all SVG's, you can convert them to gerbers or stl's using the flaps_to_pcb.sh or flaps_to_3d.py utility in this repository.
